Integración de Gstreamer

Para este proyecto hemos integrado un visualizador de vídeo en Gstreamer según la opción 2 del aula global. Sigue las siguientes instrucciones para ejecutarlo y ver un video de nuestra base de datos:

1. Abre dos terminales en tu equipo y navega hasta la carpeta en la que has guardado este programa

2. En la primera terminal, escribe: python -m SimpleHTTPServer 8888 (python3 -m http.server 8888 para python 3)

3. Desde la segunda terminal, ejecuta el pipeline de gstreamer con: sh multimedia/pipeline.sh

4. Por último, accede desde Firefox al siguiente enlace: http://localhost:8888/multimedia/index.html

Encontrarás el reproductor de video en una nueva pestaña
